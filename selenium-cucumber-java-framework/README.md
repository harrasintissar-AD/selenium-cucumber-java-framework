# Selenium Cucumber Automation Framework

## Overview
This is a professional test automation framework built using:
- Java
- Selenium WebDriver
- Cucumber BDD
- TestNG
- Page Object Model (POM)
- Maven
- Git

## Features
- Page Object Model architecture
- Cucumber BDD scenarios
- Automatic screenshots for each step
- HTML execution reports per scenario
- Clean execution folder per run
- Hooks for setup and teardown
- DriverFactory using ThreadLocal

## Test Application
https://www.saucedemo.com/

## How to Run
1. Clone the repository
2. Import as Maven project in Eclipse
3. Run `TestRunner.java` as TestNG test

## Reports
After execution:
- HTML reports: target/execution-report/
- Screenshots: generated automatically per step
